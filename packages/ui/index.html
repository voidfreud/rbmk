startup procedure
  1. cold start — core subcritical; watch ISS count-rate
  2. cock AZ bank — select AZ, hold ▲▲ until fully out
  3. select 1–4 balanced RR rods on the core map; withdraw, then pause
  4. period finite (<150 s): stop withdrawing, let power rise
  5. engage ARM (setpoint > 0) in the low-power band
  6. hand off to AR above ~5%, LAR above ~10%; hold power at setpoint

operator path: cold start → cock AZ bank → select and withdraw RR sets → engage ARM/AR → power → AZ-1 setback → AZ-5 scram → reset scram. Watch period and the ORM · PRIZMA printout; read the event log for refused actions (SEL_LIMIT, SIL_BLOK, PERIOD_BLOCK, …).

sim-only: flow slider, start at power, time accel (1×/10×/60×).

after a scram: reset scram, then restart as above — but mind the xenon: for hours after a power run the pit deepens, and ORM printouts are your only margin gauge. Sometimes the honest move is to wait it out.
